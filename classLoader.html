Test whether a block with the given name is being opened.

Parameters
nameName of the expected block.
may_failWhether it is allowed not to find the expected block.
Returns
Version number of the found block, 0 for default initialization, UINT32_MAX for failing to find the block (only if may_fail was set).
Note
If the block was not found, bytes already read of the block name are pushed back onto the stream.
