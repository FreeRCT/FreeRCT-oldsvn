enum StringTable

Table of string-parts in the game.

The largest part of the string space is allocated for 'simple' strings that exist only once, mostly in the GUI.

For rides, such as shops, this does not work since there are several shop types loaded. Instead, such strings are allocated at StringTable::STR_GENERIC_SHOP_START, while the strings of each type are elsewhere created. By using the RideType::GetString, the real string number of the queried shop type is returned.
