void FileBlock::StartSave ( const char *  blk_name,
int  version,
int  data_length 
)

Set up storing of data to the file block.

Supply name of the block, version number, and expected data length (without the 12 byte header). After setting up, use SaveUInt8, SaveUInt16, SaveUInt32, and SaveBytes to store the data in the block. Afterwards, use CheckEndSave to verify the amount of actually written data matches with the expected length.

Parameters
blk_nameName of the block (a 4 character text string).
versionVersion of the block.
data_lengthLength of the data part (that is, excluding the header).
