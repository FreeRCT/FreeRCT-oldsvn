inlinestatic

Clamp a value between an interval.

This function returns a value which is between the given interval of lower and upper. If the given value is in this interval the value itself is returned otherwise the border of the interval is returned, according which side of the interval was 'left'.

Note
The lower value must be less or equal of upper or you get some unexpected results.
Parameters
aThe value to clamp/truncate.
lowerThe minimum of the interval.
upperthe maximum of the interval.
Returns
A value between lower and upper which is closest to a.
