Read a nul-terminated string from the rcd file that takes a fixed maximum length.

Skip reading if remaining is UINT32_MAX, set it to that value if there is an error in reading.

Parameters
rcd_fileInput rcd data stream.
max_bytesMaximum number of bytes available for the string, including the nul-terminating byte.
remaining[inout] Number of bytes available in the block. UINT32_MAX means an error has happened.
Returns
The read utf-8 encoded string.
