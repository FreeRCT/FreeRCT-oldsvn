)
overridevirtual

Allow for last minute changes in the initial widget size.

If the function does nothing, you'll get the default widgets. The values BaseWidget::min_x and BaseWidget::min_y may be altered, but it may be a bad idea to make them smaller. BaseWidget::fill_x, BaseWidget::fill_y, BaseWidget::resize_x, BaseWidget::resize_y may also be changed. Since this function is being called while the window is being constructed, there is not much else that has a sane value at this point.
