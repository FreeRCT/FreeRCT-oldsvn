RideEntryResult RideInstance::EnterRide ( int  guest,
TileEdge  entry_edge 
)
pure virtual

The given guest tries to enter the ride.

Meaning and further handling of the ride visit depends on the return value.

  • If the call returns RER_REFUSED, the guest is not given entry (ride is full), it should be tried again at another time.
  • If the call returns RER_ENTERED, the guest is given access, and is now staying in the ride. The ride calls Guest::ExitRide when it is done.
  • If the call returns RER_DONE, the guest is given access, and the ride is also immediately visited (Guest::ExitRide is called before returning this answer).
    Parameters
    guestNumber of the guest entering the ride.
    entry_edgeEdge of entering the ride. Value is passed back through Guest::ExitRide, to use as parameter in RideInstance::GetExit.
    Returns
    Whether the guest is accepted, and if so, if the guest is staying inside.

Implemented in CoasterInstance, and ShopInstance.
