Get the text of a single line with an upper limit of the line length.

Text may get split on SPACE. NL implies a line-break, and NUL terminates. Note that in UTF-8, explicit tests on these characters works as expected, since the other code points all have bit 7 set of every byte they use.

Parameters
textText to search and split.
max_widthLongest allowed length of a line.
width[out] Actual width of the text.
Returns
Pointer to the end of the line, points to either a NL or a NUL.
